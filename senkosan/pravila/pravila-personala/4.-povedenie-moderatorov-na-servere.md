# 4. Поведение модераторов на сервере

{% hint style="danger" %}
Модерация не обязана знать техническую часть сервера и не должна иметь навыков кодинга.
{% endhint %}

4. Поведение модераторов на сервере

4.0 - Модератор - игрок, контролирующий поведение на сервере, оказывает помощь другим в пределах своих возможностей.

4.1 - Модератор не имеет права злоупотреблять своими полномочиями.&#x20;

* Телепортация к игрокам и самих игроков в своих целях запрещено. Исключение лишь во время модерирования или проверки игроков.
* Запрещено использование возможностей отката через coreprotect или inventoryrollback в своих целях.&#x20;

4.2 - У модерации должны быть навыки для определения читов

4.3 - Модерации запрещено разглашать приватную информацию сервера.\
&#x20;   а. Информацию из закрытых чатов\
&#x20;   b. Из решаемых/Решенных тикетов
